Three formats, one link

  • Word cloud — everyone drops one word; duplicates merge and grow. Popular words tower over one-offs, and the reveal lands with confetti.
  • Poll — facilitator-defined options, single- or multi-select, with clean bar results.
  • Q&A — open answers, upvoted by the room, sorted by votes — town-hall style.

Built for honest answers

  • Hidden until reveal (default) — the server withholds all results from everyone, facilitator included, until the reveal. No peeking, no herd effect.
  • Live progress — everyone sees "7 of 12 answered" while they wait, with a playful activity stream.
  • Question queue — the facilitator preps questions, steps through them, and earlier results archive automatically.
  • Save the cloud as an image — one click renders a PNG of the revealed word cloud to drop straight into Slack.
  • One person, one weight — repeating a word a hundred times still counts once per person.

FAQ

Is the word cloud free?

Yes — polls, word clouds, Q&A and everything else in a Scrumlo room are free, with no participant limits or paywalls.

Do participants need to install or sign up for anything?

No. They open your room link in any browser and type a display name. That's it.

Can people see results while voting?

Only if you want them to. By default results are hidden server-side until the facilitator hits Reveal — so answers can't influence each other.

Can I ask several questions in a row?

Yes — queue questions in advance, step through them, and each finished question's results are archived in the room and included in the Markdown export.

Is it anonymous?

Word cloud submissions and poll picks aren't shown with names. Rooms delete all data when they empty; nothing is logged to a database.

More in the same room

The same link also runs planning poker, sprint retrospectives, anonymous team health checks, a roadmap board, a random picker wheel and a shared timer.

Other tools keep your answers. Scrumlo keeps nothing.

A room is a single ephemeral server that deletes itself when everyone leaves. Export the results — or the cloud image — if you want to keep them. The room won't.
